# FIX Server

This project serves to provide an fake FIX server for testing.  It
is based on the QuickFIX project.  

  * What is [FIX](http://www.fixtradingcommunity.org/pg/main/what-is-fix)?

# Requirements

  * A Modern C++ compiler

  * CMake 3.0.2 or greater

  * libtool

  * automake

# Building

  1. mkdir build

  2. cd build

  3. cmake ..

  4. make
